# quickLabel
Simple Application to automate label creation from genetics db for state change tracking

MOBiLE app lives here: https://notafeature.github.io/quickLabel/quicklabel-portable.html

Note: that hosted portable build is an OLDER, reduced snapshot — it ships
only 5 workflows (ingest, agar-plate, liquid-culture, grain-spawn,
generate-batch) with no inventory / harvest / retail / swab / reprint, and
lags the main `quicklabel.html`.

**Current functionality (audited 2026-07-01):** see `FUNCTIONALITY.md` for the
complete, up-to-date map of everything the app does and how its data is stored
— the source of truth for porting this logic elsewhere.
